
# EPN-Package-System-Project

Proyecto de sistema de paquetería para la Universidad EPN de la asignatura de Programación de un grupo de estudiantes  
Este proyecto implementa un sistema de paquetería simple usando CRUD con distinción de roles entre cliente y usuario usando archivos 
para mantener una persistencia de datos.

## Estudiantes

- Diego Almeida
- Nicolay Barreno

## Tecnologías

- Lenguaje: C++
- Framework: Qt 6
- Build system: CMake
- Compilador: MinGW

## Requisitos

- Qt 6.x
- CMake 3.x o superior
- MinGW (64-bit)
- Qt Creator (recomendado)

## Compilación

Clonar el repositorio y compilar:

```bash
git clone https://github.com/reddeimer/EPN-Package-System-Project.git
```
Usar QT Creator para compilado añadiendo el archivo CMakeLists.txt y presionando  
```bash
Ctrl + R
```
